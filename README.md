# @amlplugins/mistral-files

Native `@mistralai/mistralai` SDK re-exported under the `@amlplugins` namespace.

> Mistral Files — upload, retrieve, delete files used by Fine-tuning, Batch, and Agents libraries.

## Install

```bash
npm install @amlplugins/mistral-files
```

## Use

```ts
import { /* named exports from @mistralai/mistralai */ } from "@amlplugins/mistral-files";
// or
import sdk from "@amlplugins/mistral-files";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
